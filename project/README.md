### Underlay. BGP

# Дано
Существует 2 ДЦ в каждом 3 стойки и в каждом свой кластер виртуализации PROXMOX со своими сетями и своим роутером для выхода в интернет и централизованным фаерволлом для роутинга между вланами
# Проблематика
На данный момент невозможно мигрировать ВМ из одного кластера PROXMOX в другой (из оджного ДЦ в другой) из-за того, что в каждом влане свои сети и своя адрессация. При миграции необходимо менять IP ВМ, прописывать новые правила FW для нового IP
# Цель
Получить возможность миграции ВМ
# Задача
Предложить решение для перехода на базе vxlan
